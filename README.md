# react-native-progress-bar
Simple progress bar.

`npm i @hyorax/react-native-progress-bar --save`

Simple Use
```javascript
 <ProgressBar
   percent = "53%"
   title = "ProgressBar"
   width = "100%"
   height = "100%"
   progressHeight = "30%"
   color = "red"
   backgroundColor = "#fff" 
 />                  
```
